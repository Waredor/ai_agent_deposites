# Финансовый AI-ассистент  в Telegram

Данный проект представляет собой AI-ассистента для помощи пользователю с финансовыми вопросами. Агент встроен в telegram бота.


## Структура проекта

```
ai_agent_deposites/
├── venv/                    # Виртуальное окружение
├── src/                     # Исходный код проекта и промпты
├── .env                     # Конфигурация переменных среды
├── requirements.txt         # Файл с Python-зависимостями
├── README.md                # Документация проекта
└── .gitignore               # Файл .gitignore
```

## Работа с проектом
1. Перед запуском проекта добавьте в файл .env строки вида: 
   - TG_BOT_TOKEN=ваш_токен_для_бота
   - GIGACHAT_API_TOKEN=ваш_токен_для_работы_с_gigachat

   Токен для бота можно получить с помощью @BotFather в Telegram.

   Токен для работы с GigaChat API можно получить на сайте https://developers.sber.ru/


2. Для запуска telegram бота находясь в корневой папке проекта введите команду:
      ``` 
      python run src/app.py
      ```
